## 📌 Project Overview
This project builds a Machine Learning classification model to predict the species of an Iris flower based on sepal and petal measurements.

The model is trained using the Iris dataset and deployed using a Flask web application with an interactive HTML/CSS user interface for real-time predictions.

## 🚀 Features
* Data exploration and visualization using Seaborn & Matplotlib
* Label encoding for categorical target variable
* Train-test split (80% training, 20% testing)
* Random Forest Classifier model
* Model evaluation using accuracy, precision, recall & F1-score
* Flask-based web interface for live predictions
